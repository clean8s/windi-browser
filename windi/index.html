windi-live-transform
GitHub

Use windi (and therefore, Tailwind) without any bundler.

<script src="https://unpkg.com/windi-live-transform"></script>
...
<section class="rounded shadow-xl max-w-[1523px] ...

or if you want control over where to transform:

<script type="module">
import observe from "https://unpkg.com/windi-live-transform?module"
observe(myElement)
</script>
